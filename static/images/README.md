# Images Directory
Place any local images here. The site currently uses external CDN images from the Stitch designs.

## Folder Structure
- `/images/` — General site images, OG images, favicons
- `/images/team/` — Team member photos (if you want to self-host)
- `/images/hardware/` — Arsenal product photos (if you want to self-host)
- `/images/maps/` — Map background imagery
